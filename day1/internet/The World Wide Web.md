## WWW (World Wild Web)
The internet is a huge global network that connects computers, phones, and smart devices so they can share information.

## How It Works
It works using three main things:
- **Connections** → cables (like fiber optics) and wireless signals (Wi-Fi, mobile data) that carry data
- **IP addresses** → unique IDs for each device (like a home address)
- **Protocols (TCP/IP)** → rules that make sure data is sent, routed, and received correctly

# The World Wide Web 
The World Wide Web (Web) is a system of interconnected web pages and websites that run on top of the internet.

## What It Is
- The **internet** is the network (like an ocean)
- The **Web** is the content on it (like ships on the ocean)
- Web pages are written in **HTML** and connected using **links**

## How It Works
1. You open a browser (Chrome, Firefox, etc.)
2. You enter a **URL** or click a link
3. **DNS** converts the website name into an IP address
4. Your request goes to a **server** using **TCP/IP**
5. The server sends back the page using **HTTP**
6. Your browser **renders** it and shows the page

## Key Technologies
- **HTML** → structure of the page
- **CSS** → design and style
- **JavaScript** → interactivity

> The internet is the **foundation**, and the Web is one service built on top of it (websites, pages, links).
