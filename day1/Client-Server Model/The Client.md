## Client
A **client** is the device and software you use to access the internet, like your computer or phone with a web browser.

### Key Points:
- A client sends requests to a server and receives responses.
- The most common client is a **web browser** (Chrome, Firefox, etc.).
- It loads and displays web pages using HTML, CSS, and JavaScript.
- It also handles features like tabs, cookies, and navigation.

### Other Clients:
- Email apps (Outlook, Gmail)
- Mobile apps (Instagram, TikTok)
- Games and FTP software
