Day 1: Understanding How the Web Works Day 1 of #111DaysOfLearningForChange –  Code for Change



Today I learned the fundamentals of how the web actually works 

This was a theory-focused learning session not practical implementation but it really helped me understand what happens behind every website we use.

Key takeaways:
• Difference between the Internet and the World Wide Web
  - The Internet is the global network of connected devices
  - The Web is the system of information and services built on top of it

• Client–Server Model
  - A client sends a request
  - A server processes it and sends back a response

• Core building blocks of the web
  - IP addresses help identify devices
  - DNS translates domain names into IP addresses
  - URLs locate resources on the web

• HTTP (HyperText Transfer Protocol)
  - Powers communication between client and server
  - Common methods include GET, POST, PUT, PATCH, and DELETE

Overall, it was a great reminder that every website visit depends on a coordinated system of DNS, URLs, IP addresses, and HTTP working together behind the scenes.
