# URLs (Uniform Resource Locators)

## What is a URL?
A URL is the **address of a resource on the internet**. It tells your browser where to go and how to get a specific page, image, video, or file.

Think of it like a **home address for the web** — it points exactly to one location.

![](url.png)

## Why URLs are important
Without URLs, you would need to remember complex server locations or file paths. URLs make it easy to:
- Open websites
- Share links
- Access online resources quickly

## Basic URL Structure
A typical URL looks like this:

```
https://www.example.com/page
```

It has several parts:
- **https://** → The protocol (how the browser communicates)
- **[www.example.com](http://www.example.com)** → The domain name (the website’s address)
- **/page** → The specific path (exact location of the resource)
