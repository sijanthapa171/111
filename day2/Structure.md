## HTML Structure – Short Summary
### What is Structure?
HTML structure is the **skeleton of a web page**. It organizes all elements so the browser, users, and search engines can understand the content.

---

### Basic HTML Template
Every web page follows this structure:
- `<!DOCTYPE html>` → Tells browser to use HTML5
- `<html>` → Root element of the page
- `<head>` → Contains metadata (not visible)
- `<body>` → Contains visible content
    

---

### Important Parts
**Head Section:**
- `<meta charset="UTF-8">` → Character encoding
- `<meta name="viewport">` → Responsive design
- `<title>` → Page title (tab name)
- `<link>` → Connects CSS
- `<script>` → Adds JavaScript

**Body Section:**
- Contains everything users see (text, images, links, etc.)
    

---

### Structuring Content
- Use headings (`<h1>` to `<h6>`) in order
- Use `<p>` for paragraphs
- Use `<div>` to group elements
- Use **semantic tags** for meaning:
    - `<header>` → Top section
    - `<nav>` → Navigation
    - `<main>` → Main content
    - `<article>` → Independent content
    - `<aside>` → Side content
    - `<footer>` → Bottom section
        

---

### Why Structure Matters
- Easier to read & maintain code
- Better accessibility (screen readers)
- Improves SEO
- Helps CSS & JavaScript work properly
